
# Gadget Store

A modern web application for a gadget store using .NET Web API, React.js, and PostgreSQL with JWT authentication.


## Tech Stack

**Backend:** .NET Web API

**Frontend:** React.js,Redux,Axios

**Database:** PostgreSQL

**Authentication:** JWT
